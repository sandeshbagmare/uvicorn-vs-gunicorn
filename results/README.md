# Results

This folder holds benchmark output.

- `raw/` — one JSON file per load-test run (written by `loadtest.py --out` and by `run_suite.py`).
  Each file has throughput, latency percentiles, status tally, and the per-worker-PID distribution.
- `charts/` — PNG bar charts generated by `plot_results.py` (throughput + p95 per endpoint).

Regenerate everything:

```powershell
python benchmarks\run_suite.py            # writes results/raw/*.json + prints a comparison table
python benchmarks\plot_results.py         # writes results/charts/*.png
```

Then paste the printed comparison table into the *Benchmark results* section of
[../docs/uvicorn-vs-gunicorn.md](../docs/uvicorn-vs-gunicorn.md).
